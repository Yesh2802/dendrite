# Vault Map

| Folder | Purpose |
|---|---|
| `00-inbox/` | Unprocessed captures — everything lands here first |
| `01-ideas/` | Raw ideas, sparks, half-formed thoughts |
| `02-projects/` | Active projects, each in its own subfolder |
| `03-research/` | External knowledge organized by domain |
| `04-knowledge/` | Refined, long-term concepts and mental models |
| `05-life/` | Personal planning, goals, habits |
| `06-finance/` | Financial strategy, tracking, decisions |
| `07-work/` | Professional material, career, clients |
| `08-writing/` | Long-form writing, drafts, essays |
| `meta/` | Vault system notes |
| `templates/` | Note templates for each folder type |

## Naming Conventions

- Lowercase kebab-case: `my-note-title.md`
- Project notes: `02-projects/{name}/overview.md`
- Research notes: `03-research/{domain}/{topic}.md`
